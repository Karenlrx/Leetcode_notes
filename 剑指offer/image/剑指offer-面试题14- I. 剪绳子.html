题目描述：

给你一根长度为 n 的绳子，请把绳子剪成整数长度的 m 段（m、n都是整数，n>1并且m>1），每段绳子的长度记为 k[0],k[1]...k[m] 。请问 k[0]k[1]...*k[m] 可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

以下数学推导总体分为两步：① 当所有绳段长度相等时，乘积最大。② 最优的绳段长度为 3 。

推论二： 尽可能将绳子以长度 3 等分为多段时，乘积最大。

切分规则：

  • 最优： 3 。把绳子尽可能切为多个长度为 3 的片段，留下的最后一段绳子的长度可能为 0,1,2 三种情况。
  • 次优： 2 。若最后一段绳子长度为 2 ；则保留，不再拆为 1+1。
  • 最差： 1 。若最后一段绳子长度为 1 ；则应把一份 3 + 1 替换为 2 + 2，因为。

 

算法流程：

  • 当 时，按照规则应不切分，但由于题目要求必须剪成 m>1 段，因此必须剪出一段长度为 1 的绳子，即返回 n - 1 。

  • 当 n>3时，求 n 除以 3 的 整数部分 a 和余数部分 b （即 n = 3a + b ），并分为以下三种情况：

    • 当 b = 0 时，直接返回 ；
    • 当 b = 1 时，要将一个 1 + 3 转换为 2+2，因此返回 ；
    • 当 b = 2 时，返回。

解法二：动态规划

我们也可以使用动态规划，从已知值 F(2)F(2) 逐步迭代到目标值 F(n)F(n)，它是一种自底向上的方法。 建立一维动态数组 dp：

  • 边界条件：dp[1] = dp[2] = 1，表示长度为 2 的绳子最大乘积为 1；
  • 状态转移方程：dp[i] = max(dp[i], max((i - j) * j, j * dp[i - j]))，可以这样理解：

代码演示：

func cuttingRope(n int) int {
    dp := make([]int,n+1)
    dp[1] = 1 
    dp[2] = 1
    for i := 3; i < n+1; i++ {
        for j :=1 ;  j <= i ;j++{
            dp[i] = max(dp[i], max((i - j) * j, j * dp[i - j]))// 递推公式
        }
    }
    return dp[n]
}

解法三：优化后的动态规划

我们发现任何大于 33 的数都可以拆分为数字 1，2，3 的和，且它们对 3 的余数总是 0，1，2，因此我们可以仅用 dp[0]，dp[1]，dp[2] 表示所有大于 3 的值，这样空间复杂度可降到 O(1)。

这样重复使用 dp 数组，只须一趟遍历即可完成，可使时间复杂度降到 O(N)。

 

代码演示：

xxxxxxxxxx
func cuttingRope(n int) int {
    var dp = [3]int{0,1,1} 
    for i := 3; i < n+1; i++ {
            dp[i%3] = Max(max(dp[(i - 1) % 3], i - 1),
                                            2 * max(dp[(i - 2) % 3], i - 2),
                                            3 * max(dp[(i - 3) % 3], i - 3))
        }
    return dp[n%3]
}
​
func max(i int, j int) int {
    if i > j {
        return i
    }
    return j
}
​
//因为max只能接收2个参数，所以注意重新定义一个Max
func Max(i int, j int, k int) int {
    if i > j {
        if i >k{
            return i
        }
    }else{
        if j > k {
            return j
        }
    }
    return k
}

时间复杂度：O(n) 空间复杂度：O(1)

执行用时：0 ms

内存消耗：1.9 MB
